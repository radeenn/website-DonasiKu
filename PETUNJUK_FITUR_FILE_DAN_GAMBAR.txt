PETUNJUK MENJALANKAN PROJECT DONASIKU

Project ini sudah diperbaiki untuk menangani kondisi ketika tabel documentation_files sudah ada, tetapi migration Laravel belum tercatat atau kolom title belum tersedia.

LANGKAH INSTALASI

1. Ekstrak ZIP ke folder baru agar tidak bercampur dengan project versi lama.
2. Buka CMD pada folder project.
3. Jalankan perintah berikut:

   composer install
   copy .env.example .env
   php artisan key:generate

4. Atur koneksi database pada file .env.
5. Jalankan:

   php artisan optimize:clear
   php artisan migrate
   php artisan storage:link
   npm install

6. Jalankan project menggunakan dua terminal:

   Terminal 1:
   php artisan serve

   Terminal 2:
   npm run dev

PERBAIKAN DATABASE

- Migration 2026_06_24_000016 tidak lagi mencoba membuat ulang tabel jika documentation_files sudah tersedia.
- Migration 2026_06_24_000017 akan menambahkan kolom title, file_path, file_type, created_at, dan updated_at apabila kolom tersebut belum tersedia.
- Data lama tidak dihapus.

FITUR FILE DAN GAMBAR

- Upload PDF, DOCX, PNG, JPG, JPEG, dan WEBP.
- Ukuran maksimal 5 MB.
- Preview gambar sebelum upload.
- Preview gambar dan PDF setelah disimpan.
- File DOCX dapat dibuka pada tab baru.
- File fisik ikut dihapus dari storage saat data dihapus.
- Tampilan seluruh halaman sudah direvisi tanpa emoji.

CATATAN

Jangan menjalankan php artisan migrate:fresh apabila data lama masih diperlukan karena perintah tersebut menghapus seluruh tabel.
